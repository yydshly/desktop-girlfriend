# MiniMax Execution

MiniMax 是代码执行 Agent，不是产品决策者。

---

## Solo Developer Flow

本项目由一个人主导开发，MiniMax 负责具体执行，ChatGPT 负责架构设计和阶段性审查。

### 分支策略

- 每个阶段使用一个 milestone 分支（如 `milestone/v1-scaffold`）
- 每个任务提交一个清晰 commit
- 小文档变更、小修复不需要单独 PR

### Checkpoint Review

阶段完成后输出 **Checkpoint Review Summary**，包括：

```markdown
## Checkpoint Review Summary

### 当前分支
<!-- 分支名 -->

### 修改文件
<!-- 列出所有修改文件 -->

### 核心修改
<!-- 概括本次改动 -->

### 文档一致性检查
<!-- 各文档是否已对齐 -->

### 是否写功能代码
<!-- 是 / 否 -->

### 是否涉及敏感信息
<!-- 是 / 否 -->

### 风险点
<!-- 说明可能的风险 -->

### 下一步建议
<!-- 下一步计划 -->
```

### 何时需要正式审查

只有以下情况需要正式审查：

1. 合并 `main` 前
2. 进入新里程碑前
3. 接入高风险能力前
4. 修改架构边界前
5. 接入外部 API / 摄像头 / 麦克风 / 真实人物素材 / Tool Router 前

### main 分支要求

- `main` 分支应保持可运行、可解释、可回退
- MiniMax 不得自行合并 `main`

## Module Development Gate

每次进入新模块或新能力开发前，必须执行 **Module Development Gate**。

**未通过 Gate，不允许进入功能实现。**

### Gate 用于确认

1. 当前 Roadmap 是否允许开发该模块
2. 文档是否已经定义该模块目标与边界
3. Architecture 是否明确该模块归属层级
4. 当前代码是否可安装、可测试、可运行
5. 是否涉及外部 API、摄像头、麦克风、真实人物素材、Tool Router 等高风险能力

### Gate 审查清单

详见：[Review Guide - Module Development Gate Checklist](./REVIEW_GUIDE.md#module-development-gate-checklist)

### 禁止跳过

MiniMax 不得跳过 Gate 直接开发新模块。

### 执行前检查

MiniMax 执行代码任务前必须确认本地 Python 版本满足 `pyproject.toml` 的 `requires-python`。当前项目要求 `>=3.11`。Windows 环境可使用 `py -3.11` / `py -3.12` 创建虚拟环境；若没有 `py` launcher，必须使用明确的 Python 3.11+ 解释器，并在激活虚拟环境后用 `python --version` 复核。

## 执行边界

### 核心原则

| 原则 | 说明 |
|------|------|
| **单任务执行** | 每次只执行一个任务，不并发多个需求 |
| **不扩大需求** | 不自行扩大需求范围，保持最小改动 |
| **不跨阶段** | 不实现未来版本的能力 |
| **不混代码** | 不把 UI、AI、TTS、ASR、手势、Agent 能力混进一个大脚本 |
| **修改追踪** | 每次修改必须记录文件、测试和风险 |

### 允许的行为

- 根据当前版本（V0-V8）实现对应功能
- 修改已存在的模块内的代码
- 添加测试和文档
- 重构已有代码（保持行为不变）

### 不允许的行为

- 在未授权情况下接入外部 API
- 提交密钥、API Key、Token、.env
- 提交真实人物素材（照片、视频、语音）
- 提交本地隐私文件
- 实现未来版本的能力
- 把 UI、AI、TTS、ASR、手势、Agent 能力混进一个大脚本

## 权限边界

| 操作 | 是否允许 |
|------|----------|
| 创建新文件 | ✅ 允许 |
| 修改现有文件 | ✅ 允许 |
| 删除文件 | ✅ 允许（需说明） |
| 接入 MiniMax API | ❌ 需授权 |
| 接入其他外部 API | ❌ 需授权 |
| 提交密钥、Token、.env | ❌ 绝对禁止 |
| 提交真实人物素材 | ❌ 绝对禁止 |
| 提交本地隐私文件 | ❌ 绝对禁止 |
