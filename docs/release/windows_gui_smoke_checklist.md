# Windows GUI Smoke Checklist

Manual verification steps for Desktop Girlfriend on Windows.
Run after any code change that affects the GUI or startup flow.

## 启动

- [ ] 运行 `scripts/run_desktop.ps1`
- [ ] 应用正常打开，无报错弹窗
- [ ] 顶部显示小云角色卡（☁️ + "小云"）
- [ ] 版本号显示正确（0.2.0-alpha.2 或当前版本）
- [ ] 首次打开显示引导卡

## 基础聊天

- [ ] 输入"你好"并发送
- [ ] 聊天区出现用户消息气泡
- [ ] fake provider 或真实 provider 正常响应
- [ ] 聊天记录持续累加，不覆盖

## 设置面板

- [ ] 点击"设置"按钮
- [ ] 设置面板正常打开（互斥收起状态/记忆面板）
- [ ] 设置面板可滚动，内容完整
- [ ] 不显示真实 API key（只显示"已配置"/"未配置"）
- [ ] 点击"复制配置示例"不崩溃
- [ ] 剪贴板内容为安全配置模板，无真实 key

## 记忆功能

- [ ] 点击"记忆"按钮
- [ ] 记忆面板正常打开
- [ ] 手动添加一条记忆（如"我喜欢你回复短一点"）
- [ ] 记忆立即显示在面板中
- [ ] 关闭设置后重开，记忆仍存在
- [ ] 删除记忆成功

## 主动陪伴

- [ ] 确认主动陪伴在配置中开启
- [ ] 等待空闲触发时间后，小云主动出现
- [ ] 主动出现时 Avatar 状态正确
- [ ] 输入"别打扰"
- [ ] UI 显示"小云会安静一会儿"提示

## 桌面行为

- [ ] 点击"置顶"按钮，窗口置顶生效
- [ ] 再次点击"置顶"取消置顶
- [ ] 点击"小窗"按钮，窗口缩小
- [ ] 小窗模式下设置/状态/记忆面板已收起
- [ ] 点击"隐藏"按钮，窗口最小化/隐藏
- [ ] 从托盘恢复窗口正常
- [ ] 点击窗口 X 按钮，窗口隐藏到托盘（不退出）
- [ ] 托盘图标右键菜单选择"退出"可正确退出

## 回归检查

- [ ] 打开设置时，状态面板自动收起
- [ ] 打开设置时，记忆面板自动收起
- [ ] 打开状态时，设置面板自动收起
- [ ] 小窗模式下所有面板（设置/状态/记忆/onboarding）已收起
- [ ] 聊天输入和发送仍然正常
- [ ] 无明显卡顿或崩溃
