# Prompt: 角色参考图

用途：生成统一角色参考图，后续所有 image-to-video 都以它为角色源。

## 中文提示词

```text
一个原创的温柔二次元风格桌面 AI 伙伴女孩，棕色长发，微卷发尾，侧边粉色发夹，柔和大眼睛，穿浅粉色宽松毛衣、白色百褶裙、白色袜子和室内鞋，气质亲近、安静、温暖。站在夜晚的电脑桌房间里，背景有显示器、台灯、书桌，暖色灯光，柔和景深，精致但不夸张，非写实真人，原创角色，不要任何已有 IP 或明星特征，干净画面，高质量角色设定图。
```

## English Prompt

```text
An original gentle anime-style desktop AI companion girl, long brown hair with softly curled ends, a small pink side hair clip, soft expressive eyes, wearing an oversized light pink sweater, white pleated skirt, white socks, and indoor shoes. Warm, approachable, quiet personality. Standing in a cozy night-time computer desk room with monitors, a desk lamp, books, warm lighting, soft depth of field. High-quality original character reference sheet, non-realistic, no existing IP, no celebrity likeness, clean composition.
```

## Negative Prompt

```text
existing character, celebrity, logo, watermark, text, extra fingers, extra arms, distorted face, realistic child, revealing outfit, dark horror mood, messy background, low quality
```

## 建议

- 先生成正面半身和全身各一张。
- 最终选择一张“脸最稳定、服装最清楚”的作为视频生成参考。
- 角色不要太复杂，否则视频动作时更容易变形。
