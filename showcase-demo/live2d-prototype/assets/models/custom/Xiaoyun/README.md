# Xiaoyun Custom Model Slot

This folder is reserved for the final Xiaoyun Live2D character model.

Expected files:

```text
custom/Xiaoyun/Xiaoyun.model3.json
custom/Xiaoyun/profile.json
custom/Xiaoyun/source.md
```

Setup flow:

1. Put the legal Live2D model package in this folder.
2. Copy `../profile.template.json` to `profile.json`.
3. Set `displayName` to `Xiaoyun`.
4. Use the showcase Motion Probe to map actions and expressions.
5. Select `Xiaoyun` once in the main window; the app will remember it.

Do not commit commercial or generated model assets unless the source and allowed
use are clear.
